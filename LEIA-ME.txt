================================================================
  GLOWNOW - AGENTE + CLOUDFLARE TUNNEL (URL gratuita)
================================================================

ARQUIVOS DESTA PASTA:
  - agente.py          (renomeie seu "agente_novo-3.py" para "agente.py")
  - api.py             (servidor FastAPI - NAO PRECISA EDITAR)
  - requirements.txt   (dependencias Python)
  - iniciar.bat        (rode este! ele faz tudo sozinho)

----------------------------------------------------------------
PASSO 1 - Coloque seu agente nesta pasta
----------------------------------------------------------------
1. Pegue o arquivo "agente_novo-3.py" que voce ja tem.
2. Renomeie para: agente.py
3. Coloque na MESMA pasta deste LEIA-ME.

----------------------------------------------------------------
PASSO 2 - Crie o arquivo .env (se usar Supabase)
----------------------------------------------------------------
Crie um arquivo chamado .env nesta pasta com:

    SUPABASE_KEY=sua_chave_aqui

Se nao tiver, o agente roda em modo demo (CPF de teste: 12345678900).

----------------------------------------------------------------
PASSO 3 - Rode o iniciar.bat
----------------------------------------------------------------
Clique 2x em "iniciar.bat".

Ele vai:
  [1] Verificar se Python esta instalado
  [2] Instalar o cloudflared automaticamente via winget
      (se falhar, baixe manualmente em:
       https://github.com/cloudflare/cloudflared/releases/latest
       arquivo: cloudflared-windows-amd64.exe
       renomeie para cloudflared.exe e coloque nesta pasta)
  [3] Instalar as dependencias Python (so na 1a vez, demora ~5min)
  [4] Subir a API em http://localhost:8000
  [5] Abrir o Cloudflare Tunnel

----------------------------------------------------------------
PASSO 4 - Copie a URL do Cloudflare
----------------------------------------------------------------
Vai aparecer no terminal algo tipo:

    +--------------------------------------------------------+
    |  https://abc-xyz-123.trycloudflare.com                 |
    +--------------------------------------------------------+

COPIE essa URL.

----------------------------------------------------------------
PASSO 5 - Cole no Lovable (Index.tsx)
----------------------------------------------------------------
Abra src/pages/Index.tsx e edite a linha:

    const AGENT_URL = "https://SUA-URL.trycloudflare.com/chat";

Salve. O frontend ja vai falar com seu agente!

----------------------------------------------------------------
OBSERVACOES IMPORTANTES
----------------------------------------------------------------
- Enquanto o iniciar.bat estiver aberto, o agente fica online.
- Se fechar o terminal, o site para de responder.
- A URL trycloudflare.com MUDA toda vez que voce reinicia
  (igual ao ngrok, mas sem login). Pra URL fixa de verdade,
  precisa de dominio proprio na Cloudflare.
- Para parar: feche as duas janelas de terminal.

----------------------------------------------------------------
PROBLEMAS COMUNS
----------------------------------------------------------------
"cloudflared nao e reconhecido"
  -> Reinicie o cmd/PowerShell apos a instalacao via winget.
  -> Ou baixe o .exe manual e coloque nesta pasta.

"winget nao e reconhecido"
  -> Atualize o Windows ou instale "App Installer" da Microsoft Store.

"ModuleNotFoundError: No module named 'X'"
  -> Apague o arquivo .deps_ok desta pasta e rode iniciar.bat de novo.

"Failed to fetch" no site
  -> Confira que o AGENT_URL termina em /chat
  -> Confira que o terminal do tunnel esta aberto
